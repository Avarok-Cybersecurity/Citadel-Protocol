Struct NativeIO 

pub struct NativeIO;
Expand description

Default [ProtocolIO] implementation for native platforms.

Wraps TCP/TLS/QUIC for ordered streams, UDP for unreliable datagrams, and the system RNG. All existing protocol behaviour is preserved.
