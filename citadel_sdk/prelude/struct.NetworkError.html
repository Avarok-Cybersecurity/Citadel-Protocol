Struct NetworkError 

pub struct NetworkError(/* private fields */);
Expand description

The single error type used across the entire Citadel workspace.

Pointer-sized (8 bytes): the payload is boxed so Result<T, NetworkError> stays lean on the common Ok path. Construct via the [crate::error!] macro. Derefs to [NetworkErrorInner], so err.code works directly.
